# Templates

Reusable templates for creating consistent, high-quality content across the framework.

## Available Templates

| Template | Purpose |
|----------|---------|
| [prompt-template.md](prompt-template.md) | Structure for writing new Security Copilot prompts |
| [playbook-template.md](playbook-template.md) | Structure for creating investigation playbooks |
| [use-case-template.md](use-case-template.md) | Structure for documenting new use cases |
| [agent-definition-template.md](agent-definition-template.md) | Structure for defining new multi-agent system agents |

## How to Use

1. Copy the relevant template
2. Fill in all required fields
3. Test prompts/workflows in your Security Copilot environment
4. Submit via pull request per [CONTRIBUTING.md](../CONTRIBUTING.md)
